# 🚌 projeto-aed-autocarros

Projeto de Algoritmos e Estruturas de Dados (AED) desenvolvido em Python.

Simulador de gestão de linha de autocarros utilizando:

- Listas ligadas
- Filas FIFO
- Algoritmos de ordenação
- Grafos (extra)
